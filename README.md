## Overview

This project investigates how **Generative AI has reshaped work, software development, skill requirements, and information behaviour** after the release of ChatGPT.

Using public datasets from multiple sources, the project combines **data engineering, exploratory analysis, and interactive visualization** to uncover measurable shifts in AI adoption, labour market demand, developer behaviour, and technology trends.

---

## Objectives

* Analyze the impact of Generative AI on work and software development.
* Measure changes in developer information-seeking behaviour.
* Study AI skill demand across job markets.
* Track workforce transformation after ChatGPT.
* Build an interactive analytics dashboard for exploration.

---

## Dataset

The project integrates data from **five public sources**, including:

* Stack Overflow Developer Survey
* AI-related Job Posting Dataset
* Google Trends
* Stack Overflow Trends
* Industry reports and public statistics

Overall scale:

* **138K+** developer survey responses
* **123K+** AI-related job postings
* **78-month** unified timeline

## Key Insights

* 📉 Measured a **40× decline** in Stack Overflow search interest following ChatGPT adoption.
* 🤖 Developer AI adoption increased from **44.4% → 78.5%** in two years.
* 💼 ML/AI Engineer roles require **~40× more AI-related skills** than Data Analyst roles.
* 📊 Built a unified analytical warehouse spanning **78 months** of AI-related trends.
